# Словарь проекта «Стол и Кабинет»

Единственный источник правды для слов интерфейса. **Любой текст в UI берётся
отсюда.** Расходиться коду, спецификации и прототипу больше нельзя: меняем слово —
меняем сначала здесь, потом в коде. Технический жаргон в UI запрещён (§5.5).

Название продукта — **Стол и Кабинет**.

## Территории

| Понятие | Слово в UI |
|---|---|
| Главная, место Редактора | **Стол** |
| Раздел Секретаря | **Кабинет** |
| Работа над одним фрагментом | **Тетрадь** |
| Состояние текста в момент времени | **Версия** |
