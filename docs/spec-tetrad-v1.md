# ТЗ «Тетрадь» — базовая версия (v1)

Статус: черновик на утверждение автора · 7 июля 2026
Источник: ответы автора на два опросника + дизайн-проба № 4.

## 1. Продукт одной строкой

Мастерская для тех, кто не хочет генерить тексты с ИИ, а хочет расти
как автор: работа над своим текстом с наставниками-классиками,
где последнее слово всегда за автором.

Манифест сеттинга: «У каждого писателя должен быть большой стол
и вместительный кабинет. Это их электронная версия».

## 2. Глоссарий (единые слова интерфейса)

| Слово | Значение |
|---|---|
| Стол | главный экран: текущая работа с текстами |
| Кабинет | раздел памяти: картотека, полка, голос, изыскания, аудиты |
| Тетрадь | папка одной работы: текст + версии + заметки + аудиты |
| Текст | то, что автор правит внутри тетради (слово «рукопись» в интерфейсе не используется — только в маркетинге) |
| Версия | зафиксированный снимок текста (ритуал «Зафиксировать») |
| Прописи | базовый маршрут: Не высушить → Сверить → Усилить → аудит → изыскание → финальная правка → финальный аудит |
| Проход | один этап работы инструментом/наставником |
| Депеша | промпт наставнику (в v1 — копированием в ChatGPT/Claude) |
| Диагноз | ответ наставника, разобранный на части |
| Заметки на полях | комментарии автора и наставника рядом с текстом |
| Душа текста | места, которые наставник советует не трогать (подсветка) |
| Аудит | разбор изменений за круг; финальный — первая vs финальная версия + вердикт о голосе |
| Аудит голоса | большой разбор стиля, предлагается после каждого 3-го завершённого круга |
| Роза голоса | карта стиля по осям наставников; открывается после первого аудита голоса |
| Изыскание | ресёрч; в v1 — один на круг, промпт предлагается из аудита |
| Полка | завершённые работы: финальный текст + финальный аудит (+ аудиты голоса) |
| Картотека | все тексты и их полные истории (версии, диагнозы, заметки) |
| Секретарь | голос системы: напоминания, память, подколки |
| Письмо мастеру | обратная связь разработчику |

## 3. Состав базовой версии

- 3 наставника: Чехов (чистота стиля), Соркин (драматургия), Зинссер (ясная нехудожественная проза).
- Прописи — закреплённый маршрут, этапы не настраиваются.
- Депеши копированием (текст не покидает устройство; ИИ-ключей нет).
- Цена: разовая покупка. Пробный режим: 1 текст, 1 полный круг.
- Рынок: русскоязычный.

## 4. Статусы тетради (закреплены)

| Статус | Цвет | Включается сам, когда |
|---|---|---|
| Черновик | серый #8f8776 | внесён первый текст |
| У наставника | жёлтый #d9a406 | нажато «Скопировать промпт» |
| Правка | синий #2e5fbf | нажато «Вставить ответ наставника» |
| Аудит | коричневый #7a4a1f | запущен аудит круга или финальный |
| Завершено | зелёный #2e7d44 | принят финальный аудит → «На полку» |

Статус виден цветной рамкой окна текста (и издалека — в карточках).

## 5. Прописи — маршрут круга

1. Текст внесён из буфера (лимит 5000 знаков с пробелами; счётчик под окном, мягкое предупреждение с 4500).
2. **Не высушить** — депеша → диагноз → правка.
3. **Сверить** — наставник на выбор (один из трёх) → депеша → диагноз → правка.
4. **Усилить** — депеша → диагноз → правка.
5. **Аудит круга** — что менялось и почему.
6. **Изыскание** — промпт предлагается из аудита; результат прикалывается заметкой.
7. Осмысление и финальная правка.
8. **Финальный аудит** — первая vs финальная версия + вердикт о голосе.
9. Принятие аудита → статус «Завершено» → полка.

Правила маршрута:
- Правка после каждого диагноза обязательна: 0 изменений — дальше нельзя;
  менее ~3% изменённых слов — секретарь подкалывает, но пропускает.
- Заметки на полях опциональны, но аудит читает их как движение мысли.
- Один текст можно проводить по Прописям сколько угодно раз (другой наставник и т.д.).
- Счётчик завершённых кругов общий; после каждого третьего — предложение аудита голоса (даже если круги на разных текстах).

## 6. Экран «Стол» (по дизайн-пробе № 4)

Сверху вниз:
1. Шапка ~1/6 экрана: манифест в 3 предложения + 4 кнопки-раскрывашки:
   Прописи · Наставники · Аудит · Голос (короткие описания под кнопками).
2. Окно текста — главный герой: рамка-статус, растягивается до полного
   экрана; справа от текста — поля с заметками; в тексте — подсветка
   «души текста». В заголовке окна — выбор шрифта (3 варианта, по умолчанию книжный).
3. Правая узкая колонка инструментов (открывается в работе с тетрадью):
   - Депеша: Скопировать промпт · Вставить ответ наставника (главные кнопки)
   - Проходы: Не высушить · Сверить (наставники Ч·С·З) · Усилить
   - Кабинетные: Изыскание · Аудит · Сводка секретаря
   - Текст: Зафиксировать версию · Заметки на полях · Экспорт
4. Лента «На столе»: 5±2 карточки (эта неделя → месяцы), ссылка в картотеку.
5. Длинная кнопка «Пройти в Кабинет» + малые кнопки разделов.
6. Нижняя панель: Наверх · На уровень выше · Письмо мастеру.

## 7. Депеша-цикл (ручной, v1)

1. «Скопировать промпт» → промпт в буфере, окно-инструкция
   («вставьте в ChatGPT или Claude и принесите ответ»), статус «У наставника».
2. «Вставить ответ наставника» → поле вставки → автоматический разбор
   на части: Диагноз · Что беречь (→ подсветка души текста) · Что менять.
   Статус «Правка».
3. Автор правит текст, пишет заметки; фиксирует версию — этап закрыт.

## 8. Кабинет

- **Картотека** — все тетради с историями; группировка: неделя → месяцы
  (свёрнутые, раскрываются кликом). Маркетинговая ценность: «вернись
  в прошлый год и перечитай себя».
- **Полка** — завершённые: финальный текст + финальный аудит.
- **Голос** — до первого аудита голоса: «путь к голосу» (счётчик «N из 3
  кругов» + строка секретаря после каждого круга); после — роза голоса
  с наложением аудитов.
- **Изыскания** — все ресёрчи с привязкой к тетрадям.
- **Аудиты** — письма с датами; экспорт и пересылка доступны в v1.
- В шапке Кабинета — краткое описание раздела; у каждой кнопки — одна строка о том, что внутри.

## 9. Механики удобства

- Автосохранение черновика постоянно; «Зафиксировать версию» — явный ритуал.
- Отменяемо: удаление заметки, фиксация версии, отправка на полку;
  в тексте до фиксации — обычная отмена (Ctrl+Z).
- Встреча вернувшегося: секретарь пересказывает, где автор остановился
  (этап + совет наставника + последняя своя заметка) и предлагает
  разминку на 2 минуты по оси наставника. Цитаты писателей — редкая приправа.
- Подколки лени: реплики секретаря, когда на столе >5 нетронутых черновиков
  (набор ~10 фраз, без ограничений на число текстов).
- Телефон: полноценная работа через адаптивную веб-версию (не нативное приложение).

## 10. Дизайн-паспорт

- Фон #f1e8d2 · поверхность #faf5e8 · чернила #2b2416 · линии #cfc09a
- Акцент изумруд #0e6f68 · золото #b98a2e · статусные цвета — см. §4
- Шрифты приложения: текст — PT Serif (создан для кириллицы, книжный);
  кнопки и служебное — PT Mono (архивное начертание); всё самостоятельно
  хостится, без CDN.
- Кнопки всегда отличаются от фона заливкой; на экране одна главная кнопка.
- Паттерн «главная кнопка + подчинённые» (Кабинет; Зафиксировать + экспорт).
- Ниманн — в пустых состояниях и ожиданиях (один предмет + одна остроумная
  строка); Фролов — в наградных моментах (экслибрисы наставников, корешки
  на полке, грамота аудита голоса); никогда — поверх рабочего текста.

## 12. Очередность разработки

1. Новый экран «Стол» по дизайн-пробе № 4 + дизайн-паспорт в код.
2. Статусы тетради с автопереключением.
3. Депеша-цикл с инструкцией и разбором ответа.
4. Прописи как маршрут (этапы, правило 3%, счётчик кругов).
5. Аудиты + полка/картотека + экспорт писем.
6. Встреча вернувшегося, подколки, пробный круг.
