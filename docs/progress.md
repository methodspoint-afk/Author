## Рабочие заметки для разработчика

- Данные тетрадей — JSON в `data/`; не коммитить тестовые записи
  (после ручных проверок: `git checkout -- data/`).
- Локальный просмотр: `npm run dev` → порт 3000 (в Codespace откроется сам).
- Пересборка снимка Pages: `npm run build` → `next start -p 3100` →
  обойти wget'ом в `docs/` (порт 3100 иногда держит старый процесс —
  освобождать `fuser -k 3100/tcp`).
- Проверки перед коммитом: `npm run typecheck`, `npm run test` (31 тест),
  `npm run build` — все должны быть зелёными.
