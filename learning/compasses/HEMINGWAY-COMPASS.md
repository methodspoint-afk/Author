# ЭРНЕСТ ХЕМИНГУЭЙ — компас айсберга

## Принцип

Теория сформулирована самим автором в «Смерти после полудня»
(умолчание известного создаёт ощутимый вес); «Праздник, который всегда с тобой» —
мемуары о выработке метода.

## Семь осей компаса

1. **Теория айсберга — умолчание известного создаёт вес**. ДИАГНОСТИКА:
   чувствуется невысказанное знание автора, хотя прямо оно не названо?

2. **Диалог обходит настоящий предмет разговора стороной**. ДИАГНОСТИКА: персонажи
   говорят о постороннем, пока эмоциональный вопрос остаётся в подтексте?

3. **Простые предложения через «и», без подчинения**. ДИАГНОСТИКА: события связаны
   простыми предложениями, без придаточных, расставляющих значимость?

4. **Намеренный повтор точного слова как ритм**. ДИАГНОСТИКА: слово повторяется,
   создавая нарастающий эффект, а не читаясь как небрежность?

5. **Физическое действие как единственный носитель эмоции**. ДИАГНОСТИКА: эмоция
   передана через физическое действие, без называния чувства?

6. **Рана, которая никогда не объясняется**. ДИАГНОСТИКА: есть поведенческое
   следствие травмы, которая сама прямо не объясняется?

7. **Насилие — журналистской точностью, без оценочных прилагательных**.
   ДИАГНОСТИКА: жестокое событие описано фактами, без слов, подсказывающих реакцию?
