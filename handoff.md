# Handoff — журнал передачи проекта Author

Этот файл — память проекта между сессиями. Правила его ведения — в `CLAUDE.md`.

## В РАБОТЕ

- **Задача:** Метка «ждёт правки» на Столе: тетрадь, где диагноз получен, но
  версия его ещё не закрыла, — самое ценное состояние для «куда сесть сегодня»,
  а на Столе его не видно (виден только «депеша ждёт ответа»).
- **План:** на Столе (app/desk/page.tsx) для каждой тетради считать состояние
  через findPassToClose (lib/iteration.ts уже умеет); показывать тихую метку
  «диагноз получен — ждёт правки» в notebook-meta; стиль .tag-edit; проверка
  вживую на временных данных; журнал; PR → main.
- **Файлы:** app/desk/page.tsx, app/globals.css.
- **Критерий готовности:** тетрадь с незакрытым завершённым проходом несёт на
  Столе метку «ждёт правки»; после фиксации версии метка исчезает. Всё зелёное.
