# Handoff — журнал передачи проекта Author

Этот файл — память проекта между сессиями. Правила его ведения — в `CLAUDE.md`.

## В РАБОТЕ

- **Задача:** Флоу «Новая тетрадь» на Столе — автор может завести тетрадь с нуля
  (название + первый текст фрагмента). Сейчас в UI нет способа создать тетрадь,
  только работать с уже перенесёнными.
- **План:**
  1. `createNotebook` в `app/desk/actions.ts`: валидирует название и текст,
     создаёт `Notebook` + первую `FragmentVersion`, редиректит на `/desk/[id]`.
  2. Клиентский компонент `components/NewNotebookForm.tsx` (useActionState,
     раскрытие по кнопке — как InquiryForm).
  3. Встроить кнопку/форму на `app/desk/page.tsx`.
  4. При необходимости — стили в `app/globals.css`.
  5. Тест на `createNotebook` в `tests/`.
- **Файлы:** `app/desk/actions.ts`, `components/NewNotebookForm.tsx`,
  `app/desk/page.tsx`, `app/globals.css`, `tests/notebook-create.test.ts`.
- **Критерий готовности:** на Столе есть «Новая тетрадь»; заполнив название и
  текст, автор получает тетрадь с первой версией и попадает на её страницу, где
  сразу может создать проход. Тесты и typecheck зелёные.
